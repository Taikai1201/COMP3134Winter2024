What is the definition of server port?
- A server port is a communication endpoint on a computer network, identified by a numerical value, that is used to distinguish different services or processes on the same device.

What is the server port range?
- The server port range is from 0 to 65535.

How many bits does the number above represent?
- The number 65535 represents 16 bits.
